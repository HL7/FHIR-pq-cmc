methodOrig

Codes specifying the source of the method.

This code system http://hl7.org/fhir/pq-cmc/CodeSystem/methodOrig defines the following codes:

CodeDisplayDefinition
C96102 CompendialMethod defined in any recognized compendium (e.g., USP, PharmEU, JP, etc.).
C96103 ProprietaryMethod defined by the sponsor (not recognized in CFR or any compendium)
C96164 CFRMethod defined in the Code of Federal Regulation (CFR)
