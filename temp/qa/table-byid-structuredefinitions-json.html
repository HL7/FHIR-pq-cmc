Additional Information

A textual field to provide any additional information about the acceptance criteria. [Source: SME Defined]&nbsp;Example: value changed from 4% to 5% on 01/01/2010).

PharmQualitySpecification

Describes the protocol for checking the chemical, physical and biological properties in pharmaceutical manufacturing.

Reference to Procedure

A sponsor/applicant provided alphanumeric code that describes the procedure. [Source: SME Defined] Example: HP1234-2008 Note: This could also be a transferred lab method.

Relative Retention Time

The ratio of the retention time of a component relative to that of another used as a reference obtained under identical conditions as an alias for the name of the unidentified impurities. [Source: Adapted from USP] &nbsp;Example: 1:23 (a ratio)&nbsp;Note: This is the title or name of the impurity and not the value.

Specification Status

The current FDA regulatory status of the specification. [Source: SME Defined]&nbsp;Examples: Approved, Not Approved, etc.&nbsp;Note: There are instances when FDA does approve the Specifications in a supplement or an amendment where other information in the dossier has not changed. &nbsp;Note: This is different from Application Status.

Specification Type

A classification of specification related to the kind of the entity it is referencing. [Source: SME Defined]&nbsp;Examples: Drug product, drug substance, etc.

Test Method Origin

A coded value specifying the source of the method. [Source: SME Defined]&nbsp;Example: Compendial.

Test Order

The display sequence of the test.
